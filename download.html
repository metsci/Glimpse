Glimpse Core 3.2.0

Core contains the basics. Everything you need to augment your Swing applications with amazing data visualizations. For most users, this is all you need.

Full contains extensions for integrating Glimpse with SWT, Google DSPL, and NOAA ENC charts. If you are using Maven you can depend on the various Glimpse extensions individually.
